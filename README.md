go-morse
========

A simple go program to convert strings into morse code

Usage
-----

    $ go run morse.go "This is a test"
    - ···· ·· ···        ·· ···        ·-        - · ··· -
    
or

    $ go build morse.go
    ./morse "This is a test"
    - ···· ·· ···        ·· ···        ·-        - · ··· -
